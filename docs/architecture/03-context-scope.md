## 系統邊界圖

```
┌──────────────┐     HTTPS      ┌──────────────┐
│   使用者      │ ──────────────→│    Nginx      │
│  (瀏覽器)    │                │  (SSL 終端)   │
└──────────────┘                └──────┬───────┘
                                       │
                                       ▼
                               ┌──────────────┐
                               │   maid-app    │
                               │  (Next.js     │
                               │   Docker)     │
                               └──────┬───────┘
                                       │ 內網 IP
                                       ▼
                               ┌──────────────┐
                               │  PostgreSQL   │
                               │  (VM host)    │
                               └──────────────┘
```

系統完全自給自足，無外部依賴。

## 外部介面確認結果

| 問題 | 確認結果 |
|------|---------|
| 表單資料是否同步到其他系統（CRM、ERP）？ | **否** — 資料只存在本地 DB |
| 是否有第三方服務（Email、LINE Notify、GA、Meta Pixel）？ | **否** — 無任何第三方服務 |
| `campaign.1111.com.tw` 上的 WordPress 是否繼續運行？ | **否** — WordPress 不再運行 |
| 管理員從內網或外網登入？ | **外網** — JWT 保護即可，未來可在 Nginx 層加 IP 白名單 |
| ewill-web DB 是否互通？ | **否** — 各自獨立 database，不共用資料 |

## Scope

- **包含**：前台 4 頁（首頁、服務介紹、FAQ、申請表單）+ 管理後台（登入、申請列表、Dashboard）
- **不包含**：第三方整合、自動通知、防機器人（初期不做）
